# B2B SaaS Cold Outreach Research Project

## Overview

This repository contains a structured research project on building an effective cold outreach pipeline for B2B SaaS companies.

The objective is to research experienced practitioners, collect high-quality educational content, and organize it into a reusable knowledge base that can later be converted into a practical outbound sales playbook.

---

## Project Goals

- Research leading experts in B2B SaaS cold outreach
- Collect recent YouTube transcripts and LinkedIn content
- Organize research in a structured GitHub repository
- Practice working with AI coding tools and APIs
- Build a reusable sales knowledge base

## Research Methodology

The project focuses on experienced practitioners who actively build, teach, or operate B2B SaaS outbound sales systems.

Research materials include:
